# chin

#### 介绍
一个由js编写的简便化播放器。
[查看效果](http://blog.zyczy.com.cn/demo)

#### 安装教程

1.  将所有文件clone到你的web服务器
    ```
    git clone https://gitee.com/zyc18017894/ckin.git
    git clone https://github.com/18017894/ckin.git
    ```
2.  查看demo.html
3.  按照demo.html的方式进行初始化
